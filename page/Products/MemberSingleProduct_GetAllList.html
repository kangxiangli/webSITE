单品

获取单品信息

URL

http://192.168.1.111:8011/API/Products/MemberSingleProduct/GetAllList

支持格式

JSON

HTTP请求方式

POST

是否需要登录

是

请求参数

参数 必选 类型及范围 说明
PageIndex true int 页码（默认值1）
PageSize false int 每页显示数量（默认值10）
ParentStatus true string 父级分类 （上装；下装；包；鞋子；首饰）
ColorName true false 颜色
Style true false 风格

注意事项

无

返回字段说明

返回值字段 字段类型 字段说明
ResultType int 0：输入信息验证失败；1：指定参数的数据不存在；2：操作没有引发任何变化；3：操作成功；4：操作引发错误；5：名称重复
Message string 返回信息
ProductType int 单品类型（0表示用户上传，1表示用户购买过）
ProductId int 单品Id
CoverImage string 封面图片
Status string 类型 （套头；开衫；裤子；裙子 ；连体；围巾；帽子；包；鞋子；饰品）
Price decimal 价格
Season string 季节
Size string 尺码
ColorName string 颜色
ProductType int 商品类型（0表示上传的单品，1表示购买过的单品）
IsApproval int 点赞状态（0表示登录会员自己没有点赞，1表示登录会员自己已经点赞）
ApprovalCount int 点赞数量
CommentCount int 评论数量
