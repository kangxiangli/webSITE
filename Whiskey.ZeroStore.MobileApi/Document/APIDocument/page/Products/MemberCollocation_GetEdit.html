返回字段说明

返回值字段 字段类型 字段说明
ResultType string 0：输入信息验证失败；1：指定参数的数据不存在；2：操作没有引发任何变化；3：操作成功；4：操作引发错误；5：名称重复；6、登录异常
Message string 返回信息
ColloId int 搭配Id
CollocationName string 搭配名称
ColorId int 颜色Id
ProductAttrId int 风格Id
ProductAttrName string 风格名称
SeasonId int 季节Id
SeasonName string 季节名称
SituationId int 场合Id
SituationName string 场合名称
BackGroundId int 背景素材Id
BackGroundPath string 背景素材路径
ImagePath string 搭配图路径
ImageId int 搭配图Id
ImageList array 搭配零件图
TextList array 搭配文字

ImageList
返回值字段 字段类型 字段说明
Id int 零件图Id
ProductSource int 图片来源（0、会员单品；1、商城单品；2、粉丝单品；3、上传单品；4、素材单品）
ProductType int 商品来源类型（0、用户上传；1、用户购买；）
ProductId int 商品Id
Level int 层级（可以为空）
ImagePath string 图片路径
Frame string 图片信息
Spin string 图片旋转信息

TextList

Id int 文字Id

Frame string 文字信息
Spin string 文字旋转信息
Color string 文字颜色
FontSize string 文字大小
