单品

获取单品信息

URL

http://101.200.175.177:8057/API/Products/MemberSingleProduct/Delete

支持格式

JSON

HTTP请求方式

POST

是否需要登录

是

请求参数

参数 必选 类型及范围 说明
MemberId true int 用户Id
ProductId true int 单品Id
ProductType true int 单品类型（0表示用户上传，1表示用户购买过）

注意事项

无

调用样例及调试工具

API测试工具(暂无此功能)

返回结果

JSON示例

{
    "ResultType":"3" 
    "Message":"删除成功" 
}

关于错误返回值与错误代码，参见 错误代码说明

返回字段说明

返回值字段 字段类型 字段说明
ResultType string 0：输入信息验证失败；1：指定参数的数据不存在；2：操作没有引发任何变化；3：操作成功；4：操作引发错误；5：名称重复
Message string 返回信息
